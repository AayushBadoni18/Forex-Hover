Setup Guide

Forex Hover Pro uses ExchangeRate-API v6 for live rates. The API key must be added directly into the extension source file â€” it takes about 2 minutes.

Step 1 â€” Get a free API key

Register for a free account at exchangerate-api.com. The free tier gives you 1,500 requests per month â€” more than enough for personal use. Rates are cached locally for 30 minutes, so most hovers never even hit the network. After registering, copy your key from the dashboard.

Step 2 â€” Open background.js

In your extension folder, open the file called background.js in any text editor (VS Code, Notepad, etc.). Near the top of the file you will find this line:

// TODO: Replace with your ExchangeRate-API v6 key
const API_KEY = "";

Replace the empty "" with your key inside the quotes. For example:

const API_KEY = "abc123your_actual_key_here";

Save the file.

Step 3 â€” Reload the extension

After saving background.js, go to chrome://extensions in your browser. Find Forex Hover Pro and click the â†º Reload button. The extension will pick up the new key immediately.

Then refresh any tab you want to use the extension on â€” the content script will reconnect and start converting prices.

Why is the key in the source file?

Chrome MV3 service workers start up before chrome.storage is accessible on a fresh install. Hardcoding the key directly in background.js is the most reliable way to ensure it is available the very first time the worker runs, with zero race conditions. Since this extension is for personal use and runs locally on your machine, the key never leaves your device.

About this project

About the Extension

Built to solve a real frustration. A small tool that makes global browsing feel local.

Why I built this

I built this extension because I was constantly switching tabs just to convert prices while browsing international websites. Whether I was looking at tech products, courses, or tools, I'd see a price in a different currency and immediately open another tab to check what it meant in mine. It was repetitive, distracting, and honestly unnecessary.

This extension solves that.

Now, when you hover over or select a price on any webpage, it instantly shows the converted value in your chosen currencies using live exchange rates. No extra tabs. No manual calculations. No friction.

Thoughtful edge cases

While building it, I realized that currency symbols like "$" aren't always clear â€” they could mean USD, CAD, AUD, and more. So I added a website-specific override system that lets users assign the correct base currency to specific domains. That way, conversions stay accurate even when symbols are ambiguous.

Rate provider

Live rates come from ExchangeRate-API v6. The free tier provides 1,500 requests/month â€” far more than enough for personal use. Rates are cached locally for 30 minutes, so most hovers never touch the network.

Architecture

Chrome Manifest V3 with a Service Worker as the background process. Price detection uses a TreeWalker for fast DOM traversal. The tooltip renders inside a Shadow DOM to stay isolated from host page styles. A MutationObserver with debouncing keeps dynamic pages (React, Vue, infinite scroll) fully covered.
